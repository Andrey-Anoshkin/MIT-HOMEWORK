\intro

\textit{Компьютерная игра} "--- программа, служащая для организации игрового процесса (\textit{геймплея}), связи с партнёрами по игре, или сама выступающая в качестве партнёра.

Индустрия компьютерных игр (разработка, издание и продвижение игр) "--- одна из самых быстро
развивающихся отраслей компьютерных технологий и одновременно глобального сектора
развлечений. Игры становятся культурными феноменами и признаются произведениями
искусства. Число геймеров растет, а сам гейминг становится
высокооплачиваемой профессией. Формируется мощная экосистема. Вокруг ядра, состоящего из
разработчиков игр и издателей, развиваются профильные СМИ, специальные финансовые и
рекламные инструменты; появляются специализированные физические площадки; разработке игр начинают обучать в высших
учебных заведениях и др. Кроме того, занять свою нишу хотят и игроки со смежных рынков "---
производители электроники или интернет"=корпорации,
обладающие всеми ресурсами для вывода на рынок перспективных игровых решений.\cite{intr}


\section{Разработка игр}

Разработка игр "--- это сложный многоплановый процесс, в котором участвует множество самых разнообразных специалистов.

Компьютерные игры появились во второй половине ХХ века. В дальнейшем эта
отрасль информационных технологий постоянно развивалась и сейчас превратилась в
самостоятельную индустрию. Первоначально компьютерные игры рассматривались в классическом понимании этого понятия: игра "--- это <<занятие, служащее для развлечения,
отдыха>>. Можество таких игр сейчас доступно в интернете. Лишь
позднее компьютерные игры стали применять и для обучения.\cite{duminsh}

Первые примитивные компьютерные игры работали на таких платформах, как университетские мейнфреймы и компьютеры EDSAC (\textit{Electronic Delay Storage Automatic Calculator}).

Сейчас для их создания используются всевозможные среды разработки компьютерных игр. 

\subsection{Популярные среды разработки}

Рассмотрим популярные среды разработки.

\subsubsection{\bf Unreal Engine}
	История Unreal Engine длится десятилетиями, однако сейчас он достиг своего величия. Движок разрабатывается компанией Epic Games и является передовым решением для создания крупных AAA"=игр.
	
	\begin{itemize}
	
	\item \bf Язык программирования: \normalfont C++, NoCode (Blueprints)
	
	\item \bf Стоимость: \normalfont Бесплатно (с условиями)
	
	\item \bf Плюсы \normalfont Unreal Engine:
	\begin{enumerate}
		\item Мощный редактор на все случаи жизни;
		\item Гибкая архитектура игрового движка;
		\item Игровой движок разрабатывается в том числе для игр создателя. Поэтому он в первую очередь нацелен на других разработчиков, а не бизнес, как в случае с Unity;
		\item Готовый к AAA"=проектам;
		\item Кроссплатформенный;
	\end{enumerate}
	
	\item \bf Минусы \normalfont Unreal Engine:
	
	\begin{enumerate}
		\item Более высокий порог вхождения;
		\item Более закрытое и не такое многочисленное сообщество;
		\item Акцент "--- на AAA"=проекты;
		\item Размер движка и его требовательность;
	\end{enumerate}
	
	\item \bf Примеры игр: \normalfont PUBG, Mortal Kombat, Outlast, Little Nightmares, It Takes Two
	
	\end{itemize}
	
	
	
	
	\subsubsection{\bf Source}
	Игровой движок, разработанный компанией Valve Corporation и используемый ею для создания собственных компьютерных игр.
	
	\begin{itemize}
	\item \bf Язык программирования: \normalfont C++
	
	\item \bf Плюсы \normalfont Source:
	
	\begin{enumerate}
	\item Широкие возможности использования; 
	\item Качественная графика;
	\item Низкие требования к ПК;
	\end{enumerate}
	
	\item \bf Минусы \normalfont Source:
	
	\begin{enumerate}
	\item Устаревший;
	\item Несвободное программное обеспечение;
	\end{enumerate}
	
	\item \bf Примеры игр: \normalfont Half"=Life 2, Team Fortress 2, Left 4 Dead 2, Dota 2, Counter"=Strike: Global Offensive
	
	\end{itemize}
	

	\subsubsection{\bf Unity}
	
	Кросспалтформенный движок, позволяет создавать игры под популярные ОС, такие как Windows, MacOS, Linux, Android, IOS, а также для игровой консоли Xbox.
	
	Очень популярен среди начинающих разработчиков, так как является бесплатным, функциональным и предоставляет возможность разрабатывать игры на все популярные платформы.
	
	\begin{itemize}
	\item \bf Язык программирования: \normalfont C\#, JavaScript
	
	\item \bf Стоимость: \normalfont Бесплатно
	
	\item \bf Плюсы \normalfont Unity:
	\begin{enumerate}
	\item Кроссплатформенность;
	\item Огромная библиотека настроек и плагинов;
	\item Лёгок в освоении;
	\end{enumerate}
	
	\item \bf Минусы \normalfont Unity:
	
	\begin{enumerate}
	\item Для реализации сложных проектов необходимо знание C\#;
	\item Необходимость в оптимизации;
	\item Тяжёлый вес конечного продукта;
	\end{enumerate}
	
	\item \bf Примеры игр: \normalfont Genshin Impact, The Long Dark, The Forest, Hollow Knight, Hearthstone
	
	\end{itemize}
